Want to be a data scientist? Time you invest on learning R, will not be wasted for sure. R may not be popular among the general software developers and rightly so since it is not a general purpose programming language. Instead R is loved and respected by data scientists around the world for its capabilities. <!--more-->

R is an open source programming language for statistical computing. It runs on all the major computing platforms and operating systems. The open source nature of R has enabled thousands of developers around the world to contribute and develop it. The functionality of R is enhanced by more than 10000 additional free packages available in [CRAN][cran-project] pakage repository.
 
# Installing R

The method for installing R may vary depending on your operating system. For most major Linux distributions R can be installed through their respective package managers. For (Mac) OS X and Windows binaries are available at [R project] [r-project]. Detailed procedures for installing R can be found at [R project] [r-project].

## Installing R Studio

Once R is installed it can be run as a command line application. While you may continue to do so, a serious R programmer would benefit a lot from a graphical front end. R studio which provides a code editor, console, debugging and visualizing tools is such an immensely powerfull front end which can make your journey in R much easier. The good thing is [R Studio] [r-studio] is also avialable as an open source edition. You can install R Studio by downloading the corresponding package for your platform at [R Studio donwload page] [r-studio-download].
